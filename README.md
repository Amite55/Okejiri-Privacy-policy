## 👤 User Roles

The Application supports three types of users:

- **User (Customer)**
- **Service Provider**
- **Company (with employees)**

## 📊 Information We Collect

### 1. Personal Information
We may collect:
- Full Name  
- Email Address  
- Phone Number  
- Profile Image  
- Address  

### 2. Authentication Data
- Email and Password  
- Google Sign-In information  

### 3. KYC (Know Your Customer) Information
To verify identity, we may collect:
- National ID or other verification documents  
- Personal verification data  

### 4. Location Data
- Real-time location data to enable service tracking and delivery  

### 5. Booking & Service Data
- Instant and scheduled bookings  
- Single and group bookings  
- Order history and service details  

### 6. Communication Data
- Messages between users and service providers (only during active or pending orders)  

### 7. Device & Technical Data
- Device information  
- Log data  
- IP address  

### 8. Push Notification Data
- Firebase Cloud Messaging (FCM) token for notifications  

### 9. Media Access
- Access to device gallery to upload images (profile, service proof, etc.)

## 🎯 How We Use Information

We use the collected data to:

- Provide and manage services  
- Process bookings and service requests  
- Verify user identity (KYC)  
- Enable communication between users and providers  
- Track service delivery using location data  
- Send notifications (order updates, alerts, reminders)  
- Improve app performance and user experience
